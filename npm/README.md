# dali-mcp (npm)

`npx`-friendly wrapper for [Dali by Lulu](https://dali.getlulu.dev) — connects stdio-only MCP clients to the hosted Dali server without needing Python.

```json
{
  "mcpServers": {
    "dali": { "command": "npx", "args": ["-y", "dali-mcp"] }
  }
}
```

Full docs, tools, and the self-hosted Python option: https://github.com/Lulu-The-Narwhal/dali-mcp
